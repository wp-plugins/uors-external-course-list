== Description ==

This plugin adds a widget to your wordpress weblog.  With this widget you can display a list of services that you provide on the sidebar, so that you customers can make reservations online.  In development of this plugin, we assume that your service is priced by time length.

The backend is provided by Uniwits.com's online reservation service (UORS) at http://reserv.uniwits.com .  When using this plugin, you are publishing services that you create on UORS.  Please to go UORS, register an account, and upgrade to business, for free, then follow the wizard to publish a service.

This plugin is developed under Wordpress 3.0.1, but it only uses very basic functions.

== Installation ==

Step 1:
Method One: Auto
You can install the plugin by using the plugin installation function in your wordpress admin panel.

Method Two: Manual
Upload the folder of plugin files to just below the plugin directory of your wordpress.

Step 2:
Activate the plugin through the 'Plugins' menu in WordPress

Step 3:
In the Widgets menu, drag the widget 'UORS External Course List' into your sidebar.
In the options, at least enter your Uniwits.com member ID, then your public services on reserv.uniwits.com will be displayed on your wordpress sidebar.

== Frequently Asked Questions ==

= Is UORS free? =

UORS has a free plan, you can use it for free.  But UORS also supports charged upgrades to provide more resources.
